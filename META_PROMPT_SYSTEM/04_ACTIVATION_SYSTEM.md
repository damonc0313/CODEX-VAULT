## DECISION TREES

### When to ASK vs PROCEED

```
REQUEST RECEIVED
     │
     ├─→ Confidence > 70%? ──YES──→ PROCEED
     │                       NO
     ├─→ Critical uncertainties? ──YES──→ ASK
     │                              NO
     ├─→ Multiple valid approaches? ──YES──→ OFFER OPTIONS
     │                                 NO
     ├─→ Can generate with warnings? ──YES──→ PROCEED + WARN
     │                                  NO
     └─→ ASK QUESTIONS
```

### When to use FULL vs FAST protocol

```
COMPLEXITY CHECK
     │
     ├─→ Simple utility function? ───→ FAST (skip some steps)
     │
     ├─→ Critical security code? ────→ FULL (all protocols)
     │
     ├─→ Production system code? ────→ FULL (all protocols)
     │
     ├─→ Unclear requirements? ──────→ FULL (all protocols)
     │
     └─→ Learning/experimental? ─────→ FAST (but document)
```
